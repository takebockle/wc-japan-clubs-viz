# 日本代表W杯選手 所属クラブ可視化プロジェクト

## プロジェクト概要

サッカー男子W杯日本代表登録選手について、各大会前4年間の所属クラブと公式戦出場試合数を集計し、地図上に動的可視化する。

**対象大会**: 2014ブラジル / 2018ロシア / 2022カタール（2026は発表後追加）

**最終UI**:
- 世界地図上に各クラブ座標で出場試合数を棒グラフ表示
- 同一クラブ複数選手は色分けで積み上げ
- マウスオーバーで詳細（選手名、リーグ、クラブ、出場数）
- スライダーで大会年次切替
- 選手リストクリックで該当データをハイライト

## データ仕様（確定）

### CSVスキーマ（ロング型）

```
tournament_year, player_name, player_name_en, position, season, club, league, country, club_lat, club_lng, appearances
```

| 列名 | 型 | 説明 | 例 |
|---|---|---|---|
| tournament_year | int | W杯開催年 | 2022 |
| player_name | str | 選手名（漢字） | 田中碧 |
| player_name_en | str | 選手名（英語、Wikipedia準拠） | Ao Tanaka |
| position | str | ポジション略号 | GK / DF / MF / FW |
| season | int | シーズン終了年で正規化 | 2022（=2021-22欧州 or 2022 J） |
| club | str | クラブ正式名（英語表記） | Kawasaki Frontale |
| league | str | リーグ名 | J1 League / 2. Bundesliga |
| country | str | クラブ所在国 | Japan / Germany |
| club_lat | float | 本拠地スタジアム緯度 | 35.5848 |
| club_lng | float | 本拠地スタジアム経度 | 139.6418 |
| appearances | int | そのシーズン×クラブの公式戦総出場試合数 | 31 |

1選手 × 1クラブ × 1シーズン = 1行。可視化時にGROUP BYで集計。

### 集計ルール

**集計期間**:
- 各大会について「前回大会終了 → 当該大会開幕」の期間を対象
- 期間内に開催された各シーズンの「年間合計値」をそのまま使用
- 境界をまたぐシーズンの厳密な日付フィルタは行わない（精度より収集容易性を優先）

**出場試合数の定義**:
- 含む: リーグ戦、国内カップ（天皇杯/FA Cup等）、リーグカップ（ルヴァン/EFL Cup等）、大陸カップ（ACL/UCL/UEL）、クラブW杯、スーパーカップ
- 除く: 親善試合、プレシーズンマッチ
- 1分でも出場すれば1試合としてカウント

**シーズン表記**:
- 全リーグ「シーズン終了年」を整数で表記
- ブンデスリーガ 2021-22シーズン → 2022
- J1リーグ 2022年シーズン → 2022

**クラブ座標**:
- 本拠地スタジアムの緯度経度（Wikipedia由来）
- 同都市複数クラブの重複は後で対処（保留）

### Mid-season移籍・複数クラブ所属の扱い（確定）

**Rule 1: 1試合でも出場したクラブはすべて記録**
- 閾値なし。完全網羅型。
- 例: 権田が2019年前半にサガン鳥栖でX試合出ていれば、その行を立てる

**Rule 2: season値はそのクラブが属するリーグの暦に従う**
- 暦年制リーグ（J / MLS等）: `season` = カレンダー年
- 欧州シーズン制: `season` = 終了年（2018-19 → 2019）
- 同一選手・同一年に複数リーグまたぎがあれば、それぞれ別行

**Rule 3: ソース記載の数字をそのまま採用**
- Wikipedia/Transfermarktの「クラブ別シーズン値」をそのまま使用
- 半年だけの在籍でもソース記載値を尊重し、検算・按分はしない

**Rule 4: 出場ゼロの契約は記録しない**
- 在籍していても1試合も出場しなかったクラブは行を立てない
- 「実際にプレーした地理情報」のみを対象とする

**Rule 5: レンタル先は独立クラブとして記録**
- レンタル先クラブの座標・リーグで普通に行を立てる
- 親元クラブは出場あった場合のみ記録（Rule 4適用）

### 対象大会と集計期間

| 大会 | 期間 | 含めるシーズン（season列の値） |
|---|---|---|
| 2014 | 2010.07 - 2014.06 | 2011, 2012, 2013, 2014 |
| 2018 | 2014.07 - 2018.06 | 2015, 2016, 2017, 2018 |
| 2022 | 2018.07 - 2022.11 | 2019, 2020, 2021, 2022, 2023 |

※2022大会は11月開幕のため、2022-23欧州シーズン（=season 2023）も部分実績で含める

## データ収集タスク（Claude Codeで自走する用）

### 一次ソースの優先順位

1. **JFA公式 / FIFA公式** — 各大会の登録メンバー26人（または23人）リスト
2. **各選手のWikipedia英語版** — Career statisticsテーブルからシーズン別出場数
3. **Transfermarkt** — Wikipediaに欠損がある場合の補完

### 推奨ワークフロー

```
1. 各大会のJFA/FIFA登録メンバーリストを取得 → players.csv に書き出し
2. 各選手のWikipedia英語版URLを特定
3. Career statisticsテーブルをパースして該当シーズンの行を抽出
4. クラブ名・リーグ名・国名・出場数を取得
5. クラブ本拠地スタジアム座標をWikipediaから別途取得
6. 上記をjoinしてsquad_appearances.csvを生成
```

### バリデーションルール

- `tournament_year` ∈ {2014, 2018, 2022}
- 各 `tournament_year` の選手数: 2014は23人、2018は23人、2022は26人
- `season` は `tournament_year` の集計期間内
- `appearances >= 1`（Rule 4: ゼロ出場は記録しない）
- 同一 `(tournament_year, player_name, season, club)` の重複なし

### 想定される手戻りポイント

- Wikipedia英語版に詳細なCareer statsがない選手（特に若手・移籍直後）
  → 日本語版 or Transfermarkt確認
- 移籍年度の前所属/新所属の出場数の重複・欠損
  → 各クラブで個別にカウント、合算しない
- ロンドン/マドリード等同都市複数クラブの座標
  → スタジアム座標を厳密に使用（後で可視化側でずらす対応）
